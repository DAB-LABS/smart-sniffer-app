# SMART Sniffer App Repository

Home Assistant app repository for [SMART Sniffer](https://github.com/DAB-LABS/smart-sniffer) — disk health monitoring powered by smartctl.

## Apps

| App | Description |
|---|---|
| **[SMART Sniffer Agent](smart_sniffer_agent/)** | Runs the SMART Sniffer monitoring agent on the HA host with an interactive web dashboard and mock test lab. |

## Installation

Add this repository to your Home Assistant add-on store:

```
https://github.com/DAB-LABS/smart-sniffer-app
```
